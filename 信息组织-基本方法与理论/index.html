# 信息及其相关概念

信息量的大小：消除不确定性的程度（信息论）

概念定义
数据 分散孤立对客观事物进行记录并可以鉴别的符号。
信息 具有相关性
知识 与行动相关是人们通过实践对客观事物及其运动过程和规律的认识
智慧 具有前瞻性对知识进行组合

信息是知识的原材料，是信息组织的对象。

# 信息组织的必要性

  • 人具有组织的本能
  • 组织信息因为需要检索信息
  • 组织信息有利于保存人类发展的成果
  • 失去控制和无组织的信息不再是一种资源
  • 知识和信息的海量性和无限性与人的精力、时间的有限性形成了尖锐的矛盾
  • 知识和信息的无序性和污染性与人类使用的选择性同样是尖锐的对立

# 信息资源描述

本节介绍信息资源描述的含义，重点介绍了文献著录标准化、网络信息资源描述标准化两部分的内容。

含义：又称文献著录或书目著录。根据信息织和检索的需要，对信息资源的内容特征和形式特征进行分析、选择和记录的活动。
信息描述通常是依据一定的描述规范与标准进行的

  1. 文献著录标准化
  • 著录标准
    • ISBDISBDISBD (国际标准书目著录)
    • AACR2AACR_2AACR2​ (英美编目条例第二版)
  1. 网络信息资源描述标准化
  • MARCMARCMARC (Machine Readable Catalogue) 机读目录格式的调整
  • 元数据方案DCDCDC (都柏林核心集)
    是描述网络信息资源的一种简单元数据，是文献链接的客体，它包括 15 个数据单元
# 信息资源标引

本节介绍了信息资源标引的含义与标引过程，重点从标引时间的先后、构成原理两方面介绍了标引语言的类型。

含义：也称文献标引，指分析文献的内容特征及相关外表属性，用特定语言表达分析出的属性或特征，从而赋予文献检索标识的过程。

过程：

  • 主题分析
  • 转换标识

标引语言的类型：

  1. 按标引时间的先后

    • 先组式语言（标题法 乙型肝炎 - 先天性）
    • 后组式语言（单元词语言 乙型肝炎 / 先天性）
    • 散组式语言（冒号分类法 儿童图书馆：2，61）

  2. 按构成原理：

    • 分类语言
    • 主题语言
    • 代码语言

# 信息资源的检索与利用

从广义上讲信息检索包括两个过程：

  • 信息存储：即信息的标引、加工和存储过程
  • 信息检索：即信息用户的查找过程。

从狭义上讲，信息检索仅指后一部分

  • 从信息集合中发现、查出所需信息的活动与过程
  • 本质是一个匹配的过程及用户的信息需求和信息存储的信息集合进行比较和选择的过程
