# maven 编译报错 Failed to execute goal org.apache.maven.plugins:maven-compiler-plugin:3.1:compile

见：maven 编译报错 Failed to execute goal org.apache.maven.plugins:maven-compiler-plugin:3.1:compile..........

我使用的是第一种方式：
- 在标签内加入以下

<!-- 指定maven.compiler.plugin 配置版本，解决编译问题 -->
<maven.compiler.source>1.8</maven.compiler.source>
<maven.compiler.target>1.8</maven.compiler.target>

需要注意的是如果原来的已有该标签代码，只更改数字即可，而不要在之后直接加
